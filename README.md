# CheDeSvetBot

Place Telegram Bot API token into `conf/bot_token.txt`.

State DB shall be in `res/state.db`.

Resourses should be placed in `res/products`.

### Install
```sh
pip3 install python-telegram-bot pyyaml types-PyYAML
python3 setup.py
```

### Run
```sh
python3 main.py
```

### Test
```sh
python3 test.py
```
